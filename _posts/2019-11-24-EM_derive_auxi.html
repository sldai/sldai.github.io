EM Algorithm Derivation and Auxiliary function

Introduction

We often want to estimate model parameters from observed data, and the most common method is to maximize the maximum likelihood of a statistical model, i.e. maximum likelihood estimation (MLE). But in some cases the data is incomplete, e.g. some labels are missed or some variable cannot be observed, so directly applying MLE can be quite difficult. And Expectation–maximization (EM) algorithm is proposed to maximize the likelihood with these hidden variables.

The idea of the EM algorithm to solve hidden variables is to use a iterative method. We first guess the hidden data (Expectation step), then based on the observed data and guessed hidden data we can maximize the log-likelihood and solve our model parameters (Maximization step).

Next we will see the derivation of it with a simple example.

Derivation

Suppose we weigh a batch of vehicles that leave the factory, and there two types of cars, which are labeled as 000 and 111. And we already know that the weight of each type conform the Gaussian distribution:

P(W=w∣Y=i)=G(w;μi,σi),i∈{0,1}P(W=w|Y=i) = \mathcal{G}(w;\mu_i,\sigma_i), i\in\{0,1\}P(W=w∣Y=i)=G(w;μi​,σi​),i∈{0,1}

So if we have complete data, data={(w1,i1),(w2,i2),…,(wT,iT)}\{(w_1,i_1),(w_2,i_2),\dots,(w_T,i_T)\}{(w1​,i1​),(w2​,i2​),…,(wT​,iT​)}, we can simply maximize the log-likelihood with iid assumption:

log⁡P(data)=∑t=1Tlog⁡P(W=wt,Y=it)=∑t=1T[log⁡P(Y=it)+log⁡P(W=wt∣Y=it)]\begin{aligned} \log P(data)&={\sum_{t=1}^{T}}\log{P(W=w_t,Y=i_t)}\\ &={\sum_{t=1}^{T}}\left[\log P(Y=i_t)+\log P(W=w_t|Y=i_t)\right] \end{aligned}logP(data)​=t=1∑T​logP(W=wt​,Y=it​)=t=1∑T​[logP(Y=it​)+logP(W=wt​∣Y=it​)]​

Adding the constrain ∑i=01P(Y=i)=1{\sum_{i=0}^{1}P(Y=i)=1}∑i=01​P(Y=i)=1, and using Lagrange multiplier, we can solve it:

P(Y=i)=∑t=1TI(it,i)Tμi=∑t=1TI(it,i)wt∑t=1TI(it,i)σi2=∑t=1TI(it,i)(wt−μi)2∑t=1TI(it,i)\begin{aligned} P(Y=i)&=\frac{\sum_{t=1}^{T}I(i_t,i)}{T} \\ \mu_i&=\frac{\sum_{t=1}^{T} I(i_t,i) w_t}{\sum_{t=1}^{T}I(i_t,i)} \\ \sigma^2_i&=\frac{\sum_{t=1}^{T}I(i_t,i) (w_t-\mu_i)^2}{\sum_{t=1}^{T}I(i_t,i)} \end{aligned}P(Y=i)μi​σi2​​=T∑t=1T​I(it​,i)​=∑t=1T​I(it​,i)∑t=1T​I(it​,i)wt​​=∑t=1T​I(it​,i)∑t=1T​I(it​,i)(wt​−μi​)2​​
I(x,y)=1I(x,y)=1I(x,y)=1, if x=yx=yx=y, otherwise 0.

But if all cars are covered with a black cloth, we cannot get the type. The data becomes {(w1),(w2),…,(wT)}\{(w_1),(w_2),\dots,(w_T)\}{(w1​),(w2​),…,(wT​)}. So the log-likelihood changes to:

log⁡P(data)=∑t=1Tlog⁡P(W=wt)=∑t=1Tlog⁡∑i=01P(W=wt,Y=i)\begin{aligned} \log P(data)&={\sum_{t=1}^{T}}\log{P(W=w_t)}\\ &={\sum_{t=1}^{T}}\log{{\sum_{i=0}^{1}}P(W=w_t,Y=i)} \end{aligned}logP(data)​=t=1∑T​logP(W=wt​)=t=1∑T​logi=0∑1​P(W=wt​,Y=i)​

Theoretically, we can compute gradient of this function and get maximum. However, you will find it quite difficult, since the sum is embedded in log⁡\loglog. If we can extract the sum outside log⁡\loglog, it can be much easier. Jensen’s inequality help to do this.

Jensen’s inequality

If f(x)f(x)f(x) is convex and XXX a random variable then:

E[f(X)]≥f(E[X])E[f(X)]\geq f(E[X])E[f(X)]≥f(E[X])

The equal sign holds if and only if P(x=E[x])=1P(x=E[x])=1P(x=E[x])=1, i.e. XXX is a constant.

Here I just give an example to help intuitive understanding. Suppose X is discrete and P(X=x1)=t,P(X=x2)=1−t,f(x)P(X=x_1)=t, P(X=x_2)=1-t, f(x)P(X=x1​)=t,P(X=x2​)=1−t,f(x) is convex, we can get the following picture.

And if f(x)f(x)f(x) is concave, E[f(X)]≤f(E[X])E[f(X)] \leq f(E[X])E[f(X)]≤f(E[X]).

Back to the log-likelihood. First we introduce a unknown distribution P~t(Y)\tilde{P}_t(Y)P~t​(Y) with ∑i=01P~t(Y=i)=1,0≤P~t(Y=i)≤1\sum_{i=0}^{1}\tilde{P}_t(Y=i)=1, 0\leq \tilde{P}_t(Y=i)\leq 1∑i=01​P~t​(Y=i)=1,0≤P~t​(Y=i)≤1. Given log is concave, apply Jensen's inequality:

log⁡P(data)=∑t=1Tlog⁡∑i=01P~t(Y=i)P(W=wt,Y=i)P~t(Y=i)=∑t=1Tlog⁡E[P(W=wt,Y)P~(Y)]≥∑t=1TE[log⁡P(W=wt,Y)P~(Y)]≥∑t=1T∑i=01P~t(Y=i)log⁡P(W=wt,Y=i)P~t(Y=i)\begin{aligned} \log P(data)&={\sum_{t=1}^{T}}\log{\sum_{i=0}^{1}}\tilde{P}_t(Y=i)\frac{P(W=w_t,Y=i)}{\tilde{P}_t(Y=i)}\\ &={\sum_{t=1}^{T}}\log E\left[\frac{P(W=w_t,Y)}{\tilde{P}(Y)}\right]\\ &\geq {\sum_{t=1}^{T}}E\left[\log \frac{P(W=w_t,Y)}{\tilde{P}(Y)}\right]\\ &\geq {\sum_{t=1}^{T}}{\sum_{i=0}^{1}}\tilde{P}_t(Y=i)\log\frac{P(W=w_t,Y=i)}{\tilde{P}_t(Y=i)} \end{aligned}logP(data)​=t=1∑T​logi=0∑1​P~t​(Y=i)P~t​(Y=i)P(W=wt​,Y=i)​=t=1∑T​logE[P~(Y)P(W=wt​,Y)​]≥t=1∑T​E[logP~(Y)P(W=wt​,Y)​]≥t=1∑T​i=0∑1​P~t​(Y=i)logP~t​(Y=i)P(W=wt​,Y=i)​​

Here we get a lower bound of the log-likelihood. Suppose we initialize all parameters μi,σi,P(i)\mu_i,\sigma_i,P(i)μi​,σi​,P(i), if we can continuously increase this lower bound by iterating parameters, the log-likelihood also increases. EM actually uses two steps to increase the lower bound by iteration.

E-step

In E-step, through choosing P~t(Y=i)\tilde{P}_t(Y=i)P~t​(Y=i), the lower bound increase to the log-likelihood. From Jensen's inequality, only when P(W=wt,Y)P~t(Y)\frac{P(W=w_t,Y)}{\tilde{P}_t(Y)}P~t​(Y)P(W=wt​,Y)​ is a constant, the lower bound equals the log-likelihood. So we can get the following result.

∵P(W=wt,Y=i)P~t(Y=i)=α,i={0,1}∴∑i=01P(W=wt,Y=i)=α∑i=01P~t(Y=i)=α∴P~t(Y=i)=P(W=wt,Y=i)∑i=01P(W=wt,Y=i)=P(Y=i∣W=wt)\begin{aligned} &\because \frac{P(W=w_t,Y=i)}{\tilde{P}_t(Y=i)}=\alpha, i=\{0,1\}\\ & \therefore {\sum_{i=0}^{1}}P(W=w_t,Y=i)=\alpha {\sum_{i=0}^{1}} \tilde{P}_t(Y=i)=\alpha\\ &\therefore \tilde{P}_t(Y=i)= \frac{P(W=w_t,Y=i)}{{\sum_{i=0}^{1}}P(W=w_t,Y=i)}=P(Y=i|W=w_t) \end{aligned}​∵P~t​(Y=i)P(W=wt​,Y=i)​=α,i={0,1}∴i=0∑1​P(W=wt​,Y=i)=αi=0∑1​P~t​(Y=i)=α∴P~t​(Y=i)=∑i=01​P(W=wt​,Y=i)P(W=wt​,Y=i)​=P(Y=i∣W=wt​)​

So now we can represent the lower bound as:

Q(θ,θ~)=∑t=1T∑i=01P(Y=i∣W=wt;θ~)log⁡P(W=wt,Y=i;θ)P(Y=i∣W=wt;θ~)Q(\theta,\tilde{\theta})={\sum_{t=1}^{T}}{\sum_{i=0}^{1}}P(Y=i|W=w_t;\tilde{\theta})\log\frac{P(W=w_t,Y=i;\theta)}{P(Y=i|W=w_t;\tilde{\theta})}Q(θ,θ~)=t=1∑T​i=0∑1​P(Y=i∣W=wt​;θ~)logP(Y=i∣W=wt​;θ~)P(W=wt​,Y=i;θ)​

θ\thetaθ and θ~\tilde{\theta}θ~ both represent values of parameters μi,σi,P(Y=i)\mu_i,\sigma_i,P(Y=i)μi​,σi​,P(Y=i). θ\thetaθ is fixed in this step. θ~\tilde{\theta}θ~ is what we want to change to increase Q(θ,θ~)Q(\theta,\tilde{\theta})Q(θ,θ~). Clearly if θ~=θ\tilde{\theta}=\thetaθ~=θ, the lower bound increase to the log-likelihood.

In E-step, we make the expected value of P(W=wt,Y=i;θ)P(W=w_t,Y=i;\theta)P(W=wt​,Y=i;θ) as Q(θ,θ~)Q(\theta,\tilde{\theta})Q(θ,θ~) with θ~=θ\tilde{\theta}=\thetaθ~=θ. This is why it is called expectation step.

M-step

In M-step, we optimize Q(θ,θ~)Q(\theta,\tilde{\theta})Q(θ,θ~) by adjusting θ\thetaθ with θ~\tilde{\theta}θ~ fixed, and it is much easier to optimize than P(W=wt,Y=i;θ)P(W=w_t,Y=i;\theta)P(W=wt​,Y=i;θ).

θ′=arg⁡max⁡θQ(θ,θ~)=arg⁡max⁡θ∑t=1T∑i=01P(Y=i∣W=wt;θ~)log⁡P(W=wt,Y=i;θ)P(Y=i∣W=wt;θ~)=arg⁡max⁡θ∑t=1T∑i=01P(Y=i∣W=wt;θ~)log⁡P(W=wt,Y=i;θ)\begin{aligned} \theta'&=\underset{\theta}{\arg \max}Q(\theta,\tilde{\theta}) \\ &=\underset{\theta}{\arg \max}{\sum_{t=1}^{T}}{\sum_{i=0}^{1}}P(Y=i|W=w_t;\tilde{\theta})\log\frac{P(W=w_t,Y=i;\theta)}{P(Y=i|W=w_t;\tilde{\theta})} \\ &=\underset{\theta}{\arg \max}{\sum_{t=1}^{T}}{\sum_{i=0}^{1}}P(Y=i|W=w_t;\tilde{\theta})\log{P(W=w_t,Y=i;\theta)} \end{aligned}θ′​=θargmax​Q(θ,θ~)=θargmax​t=1∑T​i=0∑1​P(Y=i∣W=wt​;θ~)logP(Y=i∣W=wt​;θ~)P(W=wt​,Y=i;θ)​=θargmax​t=1∑T​i=0∑1​P(Y=i∣W=wt​;θ~)logP(W=wt​,Y=i;θ)​

Summary

Now we can make a summary:

Input: observed data {(w1),(w2),…,(wT)}\{(w_1),(w_2),\dots,(w_T)\}{(w1​),(w2​),…,(wT​)}, the form of conditional distribution P(Y=i∣W=wt;θ~)P(Y=i|W=w_t;\tilde{\theta})P(Y=i∣W=wt​;θ~), the form of joint distribution P(W=wt,Y=i;θ)P(W=w_t,Y=i;\theta)P(W=wt​,Y=i;θ), and the maximum number of iteration steps N.

First initialize θ\thetaθ as θ0\theta_0θ0​.

For n=1 to N

E-step: θ~=θn−1\tilde{\theta}=\theta_{n-1}θ~=θn−1​, compute P(Y=i∣W=wt;θn−1)P(Y=i|W=w_t;\theta_{n-1})P(Y=i∣W=wt​;θn−1​).

M-step:

θn=arg⁡max⁡θQ(θ,θn−1)=arg⁡max⁡θ∑t=1T∑i=01P(Y=i∣W=wt;θn−1)log⁡P(W=wt,Y=i;θ)\begin{aligned} \theta_n&=\underset{\theta}{\arg \max}Q(\theta,\theta_{n-1}) \\ &=\underset{\theta}{\arg \max}{\sum_{t=1}^{T}}{\sum_{i=0}^{1}}P(Y=i|W=w_t;{\theta_{n-1}})\log{P(W=w_t,Y=i;\theta)} \end{aligned}θn​​=θargmax​Q(θ,θn−1​)=θargmax​t=1∑T​i=0∑1​P(Y=i∣W=wt​;θn−1​)logP(W=wt​,Y=i;θ)​

Though the example is simple, but we can generate it to any other model. Just replace WWW with all observed variable, and replace YYY with all hidden variable.

Auxiliary Function

In E-step, we fix model parameters, and optimize the conditional distribution of hidden data. And in M-step, what we do is to fix the distribution of hidden data and optimize the values of model parameters. This idea is called auxiliary functions in mathematics.

Q(x,y)Q(x,y)Q(x,y) is an auxiliary function for f(x)f(x)f(x), if it satisfies:

(i) Q(x,x)=f(x)Q(x,x)=f(x)Q(x,x)=f(x)

(ii) Q(x,y)≥f(x)Q(x,y)\geq f(x)Q(x,y)≥f(x)

Recall the property of Q(θ,θ~)Q(\theta,\tilde{\theta})Q(θ,θ~), what we do is to construct an auxiliary function for −P(W=wt,Y=i;θ)-P(W=w_t,Y=i;\theta)−P(W=wt​,Y=i;θ), since:

(i) −Q(θ,θ)=−P(W=wt,Y=i;θ)-Q(\theta,{\theta})=-P(W=w_t,Y=i;\theta)−Q(θ,θ)=−P(W=wt​,Y=i;θ)

(ii) −Q(θ,θ~)≥−P(W=wt,Y=i;θ)-Q(\theta,\tilde{\theta}) \geq -P(W=w_t,Y=i;\theta)−Q(θ,θ~)≥−P(W=wt​,Y=i;θ)

The update rule of an auxiliary function is xn+1=arg⁡max⁡xQ(x,xn)x_{n+1}={\arg \max}_xQ(x,x_{n})xn+1​=argmaxx​Q(x,xn​).
